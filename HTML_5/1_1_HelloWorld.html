Введение

HTML

(Hypertext Markup Language) использует систему разметки, состоящую из элементов, которые представляют конкретное содержимое. Разметка означает, что в HTML вы указываете, что будет представлено зрителю, а не как это будет представлено. Визуальные представления определяются каскадными таблицами стилей (CSS) и реализуются браузерами. Все еще существующие элементы, позволяющие это делать , такие как, например, font , "полностью устарели и не должны использоваться авторами"[1].

HTML иногда называют языком программирования, но в нем нет логики, поэтому он является языком разметки. HTML-теги обеспечивают семантический смысл и машиночитаемость содержимого страницы.

Элемент обычно состоит из открывающего тега (<имя_элемента>), закрывающего тега (</имя_элемента>), которые содержат имя элемента, заключенное в угловые скобки, и содержимое между ними:

<имя_элемента>...содержимое... </имя_элемента>

Существуют элементы HTML, которые не имеют закрывающего тега или какого-либо содержимого. Такие элементы называются пустыми.

К пустым элементам относятся

<img>, <meta>, <link>, <input>

Имена элементов можно рассматривать как описательные ключевые слова для содержимого, которое они содержат, например, video, audio, table, footer

HTML-страница может состоять из сотен элементов, которые затем считываются веб-браузером, интерпретируются и выводятся на экран в виде читаемого или воспринимаемого человеком содержимого.

Для данного документа важно отметить различие между элементами и тегами:

Элементы: video, audio, table, footer
Теги: <video>, <audio>, <table>, <footer>, </html>, </body>

Понимание элементов

Давайте разберемся с тегом...

Тег <p> представляет собой обычный абзац

Элементы обычно имеют открывающий и закрывающий теги. Открывающий тег содержит имя элемента в угловом формате Скобки (<p>). Закрывающий тег идентичен открывающему тегу с добавлением прямой косой черты (/) между открывающей скобкой и именем элемента (</p>).

Контент может располагаться между этими двумя тегами:

<p> Это простой параграф </p>

Создание простой страницы

Следующий пример HTML создает простую веб-страницу " Привет Мир ".

HTML файл можно создать с помощью любого текстового редактора . Для распознавания HTML файла необходимо сохранять их с расширением .html или .htm[2].

После создания этот файл можно открыть в любом веб-браузере.

Пример кода Отображение кода
<!DOCTYPE html>
<html lang="en">

<head>
 <meta charset="UTF-8">
 <title>Hello!</title>
</head>

<body>
 <h1>Привет Мир!</h1>
 <p>Это простой параграф.</p>
</body>

</html>

Простая разбивка страниц

Это теги, используемые в примере:

Тег Значение
<!DOCTYPE>

Определяет версию HTML, используемую в документе. В данном случае это HTML5. Дополнительные сведения см. в теме "doctypes".

<html>

Открывает страницу. После закрывающего тега ( </html>) не должно быть никакой разметки. Атрибут lang указывает основной язык страницы, используя коды языков ISO (en - английский). Дополнительные сведения см. в разделе Content Language.

<head>

Открывает раздел head, который не отображается в главном окне браузера, а в основном содержит информацию о HTML-документе, называемую метаданными. Он также может содержать импорты из внешних таблиц стилей и скриптов. Закрывающим тегом является </head>.

<meta>

Предоставляет браузеру некоторые метаданные о документе. Атрибут charset определяет кодировку символов. Современные HTML-документы всегда должны использовать UTF-8 , хотя это и не является обязательным условием. В HTML тег <meta> не требует закрывающего тега. Дополнительную информацию см. в разделе "Мета".

<title>

Заголовок страницы. Текст, написанный между этим открывающим и закрывающим тегами (</title>), будет отображаться на вкладке страницы или в строке заголовка браузера.

<body>

Открывает часть документа, отображаемую пользователям, т.е. все видимое или слышимое содержимое страницы. После закрывающего тега </body> не следует добавлять никакого содержимого.

<h1>

Заголовок первого уровня для этой страницы. Дополнительные сведения см. в разделе "headings".

<p>

Представляет собой обычный абзац текста
